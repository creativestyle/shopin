# @demo/mocked-payment-service-provider

⚠️ **DEMO ONLY** — For demonstration only; not for production.

React components that simulate payment flows in the storefront (e.g. success/failure buttons). Used by the demo payment step in the presentation app.

## What’s included

- `PaymentDemo` - A demo payment page component with two buttons that simulate payment options

## Usage

The component is automatically included in the presentation app for demo purposes. It is used in the payment demo page at `/demo/mocked-payment-step/[paymentId]`.
