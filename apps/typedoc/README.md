# @apps/typedoc

Typedoc-generated API documentation site for SHOPin contracts and types.

## Scripts

- **`npm run build`** — Generate docs (output in `docs/`)
- **`npm run dev`** — Watch and rebuild on changes, serve at http://localhost:5000
- **`npm run serve-docs`** — Serve the built `docs/` folder

See the [root README](../../README.md) for monorepo setup.
