# @apps/storybook

Storybook app for UI component documentation and development (Next.js integration).

## Scripts

- **`npm run dev`** — Start Storybook dev server (http://localhost:6006)
- **`npm run build`** — Build static Storybook output
- **`npm run lint`** / **`npm run format`** — Lint and format

See the [root README](../../README.md) for monorepo setup.
